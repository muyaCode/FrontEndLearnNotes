еүҚз«Ҝж•°жҚ®иҜ·жұӮж–№ејҸдё»иҰҒеҢ…жӢ¬XMLHttpRequestгҖҒFetchгҖҒAxiosгҖҒWebSocketзӯүгҖӮиҝҷдәӣиҜ·жұӮж–№ејҸеҗ„жңүзү№зӮ№пјҢйҖӮз”ЁдәҺдёҚеҗҢзҡ„еңәжҷҜгҖӮжң¬ж–Үе°Ҷд»Һз»јеҗҲжҖ§иғҪгҖҒдјҳзјәзӮ№гҖҒжңҖдҪідҪҝз”ЁеңәжҷҜд»ҘеҸҠдҪҝз”Ёж–№ејҸзҡ„и§’еәҰеҜ№иҝҷдәӣж•°жҚ®иҜ·жұӮж–№ејҸиҝӣиЎҢеҲҶжһҗгҖӮ

XMLHttpRequestпјҲXHRпјү вҖӢ

XMLHttpRequest жҳҜеүҚз«ҜжңҖж—©дҪҝз”Ёзҡ„ж•°жҚ®иҜ·жұӮж–№ејҸгҖӮе®ғж”ҜжҢҒејӮжӯҘиҜ·жұӮпјҢеҸҜд»ҘйҖҡиҝҮи®ҫзҪ®еӣһи°ғеҮҪж•°еӨ„зҗҶиҜ·жұӮе®ҢжҲҗеҗҺзҡ„ж•°жҚ®гҖӮ

жҖ§иғҪпјҡXHR еңЁиҫғж—©зҡ„жөҸи§ҲеҷЁдёӯиЎЁзҺ°иүҜеҘҪпјҢдҪҶйҡҸзқҖжөҸи§ҲеҷЁжҖ§иғҪзҡ„жҸҗеҚҮпјҢе…¶жҖ§иғҪз“¶йўҲйҖҗжёҗеҮёжҳҫгҖӮзӣёиҫғдәҺе…¶д»–иҜ·жұӮж–№ејҸпјҢXHR зҡ„жҖ§иғҪзЁҚйҖҠдёҖзӯ№гҖӮ вҖӢ

дјҳзӮ№пјҡ вҖӢ

  • е№ҝжіӣзҡ„жөҸи§ҲеҷЁж”ҜжҢҒпјҡе°Ҫз®ЎзҺ°д»ЈжөҸи§ҲеҷЁжӣҙжҺЁиҚҗдҪҝз”Ё Fetch APIпјҢдҪҶ XMLHttpRequest д»Қз„¶еҫ—еҲ°дәҶеҮ д№ҺжүҖжңүжөҸи§ҲеҷЁзҡ„ж”ҜжҢҒпјҢеҢ…жӢ¬дёҖдәӣиҫғж—§зҡ„зүҲжң¬гҖӮ
  • дәӢд»¶й©ұеҠЁпјҡXMLHttpRequest дҪҝз”ЁдәӢд»¶й©ұеҠЁжЁЎеһӢпјҢжҸҗдҫӣдәҶ onreadystatechangeгҖҒonloadгҖҒonerror зӯүдәӢд»¶пјҢеҸҜд»Ҙж–№дҫҝең°зӣ‘еҗ¬иҜ·жұӮзҡ„дёҚеҗҢйҳ¶ж®өгҖӮ
  • ж”ҜжҢҒдёҠдј иҝӣеәҰзӣ‘жҺ§пјҡXMLHttpRequest жҸҗдҫӣдәҶ upload еұһжҖ§пјҢеҸҜд»Ҙз”ЁжқҘзӣ‘жҺ§ж–Үд»¶дёҠдј зҡ„иҝӣеәҰгҖӮ
  • ж”ҜжҢҒиҜ·жұӮе’Ңе“Қеә”еӨҙи®ҝй—®пјҡеҸҜд»ҘйҖҡиҝҮ setRequestHeader е’Ң getResponseHeader ж–№жі•жқҘи®ҫзҪ®е’ҢиҺ·еҸ–иҜ·жұӮе’Ңе“Қеә”зҡ„еӨҙдҝЎжҒҜгҖӮ
  • ж”ҜжҢҒи¶…ж—¶и®ҫзҪ®пјҡеҸҜд»ҘйҖҡиҝҮ timeout еұһжҖ§и®ҫзҪ®иҜ·жұӮзҡ„и¶…ж—¶ж—¶й—ҙпјҢе№¶еңЁи¶…ж—¶еҗҺи§ҰеҸ‘ ontimeout дәӢд»¶гҖӮ
  • ж”ҜжҢҒеҗҢжӯҘиҜ·жұӮпјҡиҷҪз„¶дёҚжҺЁиҚҗпјҢдҪҶ XMLHttpRequest ж”ҜжҢҒеҗҢжӯҘиҜ·жұӮпјҢиҝҷеңЁжҹҗдәӣзү№е®ҡзҡ„еңәжҷҜдёӢеҸҜиғҪжңүз”ЁгҖӮ

  • еҹәдәҺеӣһи°ғпјҡXMLHttpRequest дҪҝз”Ёеӣһи°ғеҮҪж•°жқҘеӨ„зҗҶе“Қеә”пјҢиҝҷеҸҜиғҪеҜјиҮҙеӣһи°ғең°зӢұпјҲcallback hellпјүпјҢд»Јз Ғйҡҫд»Ҙз»ҙжҠӨгҖӮ

  • еҹәдәҺ PromiseпјҡFetch API иҝ”еӣһ PromisesпјҢиҝҷдҪҝеҫ—ејӮжӯҘж“ҚдҪңжӣҙеҠ жҳ“дәҺз®ЎзҗҶе’Ңй“ҫејҸи°ғз”ЁгҖӮ
  • еҶ…зҪ®зҡ„й”ҷиҜҜеӨ„зҗҶпјҡеҪ“зҪ‘з»ңиҜ·жұӮеҮәзҺ°й—®йўҳж—¶пјҢFetch API дјҡиҝ”еӣһдёҖдёӘеёҰжңүй”ҷиҜҜзҠ¶жҖҒзҡ„ PromiseпјҢеҸҜд»Ҙж–№дҫҝең°дҪҝз”Ё .catch() ж–№жі•иҝӣиЎҢеӨ„зҗҶгҖӮ
  • жөҒејҸе“Қеә”пјҡFetch API ж”ҜжҢҒжөҒејҸе“Қеә”пјҢиҝҷж„Ҹе‘ізқҖдҪ еҸҜд»ҘеӨ„зҗҶжӯЈеңЁдёӢиҪҪзҡ„ж•°жҚ®пјҢиҖҢдёҚеҝ…зӯүеҫ…ж•ҙдёӘе“Қеә”дҪ“дёӢиҪҪе®ҢжҲҗгҖӮ
  • и·Ёеҹҹж”ҜжҢҒпјҡFetch API еӨ©з„¶ж”ҜжҢҒ CORSпјҲи·Ёжәҗиө„жәҗе…ұдә«пјүпјҢдҪҝеҫ—и·ЁеҹҹиҜ·жұӮжӣҙеҠ е®№жҳ“е®һзҺ°гҖӮ
  • иҜ·жұӮе’Ңе“Қеә”еҜ№иұЎпјҡFetch API жҸҗдҫӣдәҶ Request е’Ң Response еҜ№иұЎпјҢиҝҷдәӣеҜ№иұЎеҸҜд»Ҙи®©дҪ жӣҙе®№жҳ“ең°жҺ§еҲ¶иҜ·жұӮзҡ„иЎҢдёәе’Ңи®ҝй—®е“Қеә”зҡ„еҶ…е®№гҖӮ

зјәзӮ№пјҡ вҖӢ

  • й»ҳи®ӨдёҚжҗәеёҰ CookieпјҡFetch API еңЁй»ҳи®Өжғ…еҶөдёӢдёҚдјҡеҸ‘йҖҒеҗҢжәҗзҡ„ CookieпјҢиҝҷеҸҜиғҪеҜјиҮҙдёҖдәӣеҹәдәҺ Cookie зҡ„и®ӨиҜҒжңәеҲ¶еҮәзҺ°й—®йўҳгҖӮеҸҜд»ҘйҖҡиҝҮи®ҫзҪ®иҜ·жұӮзҡ„ credentials йҖүйЎ№жқҘи§ЈеҶіиҝҷдёӘй—®йўҳгҖӮ
  • дёҚж”ҜжҢҒи¶…ж—¶еӨ„зҗҶпјҡFetch API жң¬иә«дёҚжҸҗдҫӣиҜ·жұӮи¶…ж—¶зҡ„еҠҹиғҪгҖӮдёҚиҝҮпјҢеҸҜд»ҘйҖҡиҝҮеҢ…иЈ… Promise жқҘе®һзҺ°и¶…ж—¶йҖ»иҫ‘гҖӮ
  • ж—§жөҸи§ҲеҷЁе…је®№жҖ§пјҡFetch API еңЁдёҖдәӣж—§зүҲжң¬зҡ„жөҸи§ҲеҷЁдёӯдёҚиў«ж”ҜжҢҒпјҢеҸҜиғҪйңҖиҰҒдҪҝз”Ё polyfillгҖӮ
  • й”ҷиҜҜеӨ„зҗҶдёҚеӨҹзӣҙи§ӮпјҡFetch API дёҚдјҡе°Ҷ HTTP зҠ¶жҖҒз Ғдёә 4xx жҲ– 5xx зҡ„е“Қеә”и§Ҷдёәй”ҷиҜҜпјҢиҝҷж„Ҹе‘ізқҖдҪ йңҖиҰҒеңЁ .then() ж–№жі•дёӯжүӢеҠЁжЈҖжҹҘе“Қеә”зҠ¶жҖҒгҖӮ
  • еҸ–ж¶ҲиҜ·жұӮйңҖиҰҒйўқеӨ–зҡ„ APIпјҡиҷҪз„¶ Fetch API жң¬иә«дёҚж”ҜжҢҒеҸ–ж¶ҲиҜ·жұӮпјҢдҪҶеҸҜд»ҘйҖҡиҝҮз»“еҗҲдҪҝз”Ё AbortController жқҘе®һзҺ°гҖӮ
  • дёҠдј иҝӣеәҰзӣ‘жҺ§пјҡFetch API дёҚжҸҗдҫӣдёҠдј иҝӣеәҰзҡ„зӣ‘жҺ§пјҢиҖҢ XMLHttpRequest ж”ҜжҢҒгҖӮ

жңҖдҪідҪҝз”ЁеңәжҷҜпјҡзҺ°д»ЈжөҸи§ҲеҷЁдёӯпјҢйңҖиҰҒз®ҖжҙҒиҜӯжі•е’Ңй“ҫејҸи°ғз”Ёзҡ„еңәжҷҜгҖӮ вҖӢ

дҪҝз”Ёж–№ејҸпјҡ вҖӢ

  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

Axios вҖӢ

Axios жҳҜдёҖдёӘеҹәдәҺ Promise зҡ„ HTTP е®ўжҲ·з«ҜпјҢз”ЁдәҺжөҸи§ҲеҷЁе’Ң Node.jsгҖӮе®ғжү©еұ•дәҶ Fetch APIпјҢжҸҗдҫӣдәҶжӣҙдё°еҜҢзҡ„еҠҹиғҪгҖӮ

жҖ§иғҪпјҡAxios еңЁзҺ°д»ЈжөҸи§ҲеҷЁдёӯжҖ§иғҪиҫғеҘҪпјҢдёҺ Fetch зӣёеҪ“гҖӮ вҖӢ

дјҳзӮ№пјҡ вҖӢ

  • еҹәдәҺ Promise зҡ„ APIпјҡAxios дҪҝз”Ё PromiseпјҢдҪҝеҫ—ејӮжӯҘж“ҚдҪңжӣҙеҠ з®ҖжҙҒе’Ңжҳ“дәҺз®ЎзҗҶпјҢж”ҜжҢҒ .then е’Ң .catch ж–№жі•гҖӮ
  • жӢҰжҲӘеҷЁж”ҜжҢҒпјҡAxios е…Ғи®ёдҪ ж·»еҠ иҜ·жұӮе’Ңе“Қеә”жӢҰжҲӘеҷЁпјҢиҝҷдәӣжӢҰжҲӘеҷЁеҸҜд»ҘеңЁиҜ·жұӮеҸ‘йҖҒд№ӢеүҚжҲ–е“Қеә”еҲ°иҫҫд№ӢеүҚиҝӣиЎҢиҮӘе®ҡд№үеӨ„зҗҶгҖӮ
  • иҪ¬жҚўиҜ·жұӮж•°жҚ®е’Ңе“Қеә”ж•°жҚ®пјҡAxios е…Ғи®ёдҪ еңЁиҜ·жұӮеҸ‘йҖҒд№ӢеүҚиҪ¬жҚўиҜ·жұӮж•°жҚ®пјҲtransformRequestпјүе’ҢеңЁе“Қеә”еҲ°иҫҫд№ӢеүҚиҪ¬жҚўе“Қеә”ж•°жҚ®пјҲtransformResponseпјүгҖӮ
  • еҸ–ж¶ҲиҜ·жұӮпјҡAxios ж”ҜжҢҒеҸ–ж¶ҲиҜ·жұӮпјҢйҖҡиҝҮ CancelToken е®һзҺ°пјҢиҝҷжҳҜ XMLHttpRequest жүҖзјәд№Ҹзҡ„зү№жҖ§гҖӮ
  • иҮӘеҠЁиҪ¬жҚў JSON ж•°жҚ®пјҡAxios дјҡиҮӘеҠЁе°Ҷ JavaScript еҜ№иұЎиҪ¬жҚўдёә JSON еӯ—з¬ҰдёІеҪ“еҸ‘йҖҒиҜ·жұӮпјҢе№¶е°Ҷе“Қеә”дёӯзҡ„ JSON ж•°жҚ®иҮӘеҠЁиҪ¬жҚўдёә JavaScript еҜ№иұЎгҖӮ
  • е®ўжҲ·з«Ҝж”ҜжҢҒйҳІеҫЎ XSRFпјҡAxios жҸҗдҫӣдәҶйҳІеҫЎ XSRFпјҲи·Ёз«ҷиҜ·жұӮдјӘйҖ пјүзҡ„еҠҹиғҪгҖӮ
  • й”ҷиҜҜеӨ„зҗҶпјҡAxios жҸҗдҫӣдәҶз»ҹдёҖзҡ„й”ҷиҜҜеӨ„зҗҶжңәеҲ¶пјҢеҪ“иҜ·жұӮеӨұиҙҘж—¶пјҢдјҡеңЁ .catch дёӯжҚ•иҺ·еҲ°й”ҷиҜҜгҖӮ
  • еҲӣе»әе®һдҫӢпјҡAxios е…Ғи®ёеҲӣе»әе®һдҫӢпјҢе№¶еңЁе®һдҫӢдёҠи®ҫзҪ®й»ҳи®Өй…ҚзҪ®пјҢиҝҷеҜ№дәҺеӨҡж¬ЎиҜ·жұӮдҪҝз”ЁзӣёеҗҢй…ҚзҪ®йқһеёёжңүз”ЁгҖӮ

зјәзӮ№пјҡ вҖӢ

  • йўқеӨ–зҡ„дҫқиө–пјҡдҪҝз”Ё Axios ж„Ҹе‘ізқҖдҪ зҡ„йЎ№зӣ®е°Ҷдҫқиө–дәҺдёҖдёӘз¬¬дёүж–№еә“пјҢиҝҷеҸҜиғҪдјҡеўһеҠ йЎ№зӣ®зҡ„еӨҚжқӮжҖ§гҖӮ
  • жөҸи§ҲеҷЁе…је®№жҖ§пјҡиҷҪз„¶ Axios ж”ҜжҢҒеӨ§еӨҡж•°зҺ°д»ЈжөҸи§ҲеҷЁпјҢдҪҶдёҚж”ҜжҢҒ IE9 еҸҠд»ҘдёӢзүҲжң¬пјҢеҰӮжһңдҪ йңҖиҰҒж”ҜжҢҒиҝҷдәӣж—§жөҸи§ҲеҷЁпјҢеҸҜиғҪйңҖиҰҒеј•е…ҘйўқеӨ–зҡ„ polyfillгҖӮ
  • дҪ“з§ҜпјҡзӣёжҜ”дәҺеҺҹз”ҹ Fetch APIпјҢAxios зҡ„дҪ“з§ҜжӣҙеӨ§пјҢеҰӮжһңдҪ еҸӘдҪҝз”Ёе®ғзҡ„йғЁеҲҶеҠҹиғҪпјҢеҸҜиғҪдјҡж„ҹи§үдёҚеӨҹзІҫз®ҖгҖӮ
  • еӯҰд№ жӣІзәҝпјҡеҜ№дәҺеҲқеӯҰиҖ…жқҘиҜҙпјҢAxios жҸҗдҫӣдәҶеҫҲеӨҡй«ҳзә§еҠҹиғҪпјҢиҝҷеҸҜиғҪйңҖиҰҒдёҖдёӘеӯҰд№ жӣІзәҝжқҘзҗҶи§Је’ҢжӯЈзЎ®дҪҝз”Ёе®ғд»¬гҖӮ
  • зӨҫеҢәе’Ңз»ҙжҠӨпјҡиҷҪз„¶ Axios еҫҲеҸ—ж¬ўиҝҺпјҢдҪҶе®ғзҡ„з»ҙжҠӨе’Ңжӣҙж–°йҖҹеәҰеҸҜиғҪдёҚеҰӮдёҖдәӣе®ҳж–№зҡ„ API еҝ«пјҢиҖҢдё”зӨҫеҢәж”ҜжҢҒд№ҹеҸҜиғҪжңүйҷҗгҖӮ

жңҖдҪідҪҝз”ЁеңәжҷҜпјҡйңҖиҰҒеңЁйЎ№зӣ®дёӯиҝӣиЎҢеӨ§йҮҸ HTTP иҜ·жұӮпјҢдё”йңҖиҰҒдё°еҜҢй…ҚзҪ®е’ҢеҸ–ж¶ҲиҜ·жұӮеҠҹиғҪзҡ„еңәжҷҜпјӣжҳ“дёҺ Vue.js йӣҶжҲҗгҖӮ вҖӢ

axios.get('https://xxx')
  .then(response => {
    console.log(response.data);
  })
  .catch(error => {
    console.log('Error:', error);

WebSocket вҖӢ

WebSocket жҳҜдёҖз§ҚзҪ‘з»ңйҖҡдҝЎеҚҸи®®пјҢж”ҜжҢҒе…ЁеҸҢе·ҘйҖҡдҝЎгҖӮеңЁеүҚз«ҜпјҢWebSocket дё»иҰҒз”ЁдәҺдёҺеҗҺз«Ҝе®һж—¶дәӨдә’ж•°жҚ®гҖӮ

жҖ§иғҪпјҡWebSocket жҸҗдҫӣдәҶжҢҒз»ӯзҡ„ TCP иҝһжҺҘпјҢеҮҸе°‘дәҶиҜ·жұӮж¬Ўж•°пјҢжҸҗй«ҳдәҶжҖ§иғҪгҖӮ вҖӢ

дјҳзӮ№пјҡ вҖӢ

  • е®һж—¶йҖҡдҝЎпјҡWebSocket жҸҗдҫӣдәҶе®һж—¶еҸҢеҗ‘йҖҡдҝЎзҡ„иғҪеҠӣпјҢжңҚеҠЎеҷЁеҸҜд»ҘйҡҸж—¶еҗ‘е®ўжҲ·з«ҜеҸ‘йҖҒж¶ҲжҒҜпјҢиҝҷеҜ№дәҺйңҖиҰҒе®һж—¶жӣҙж–°зҡ„еә”з”ЁпјҲеҰӮе®һж—¶иҒҠеӨ©гҖҒжёёжҲҸгҖҒе®һж—¶ж•°жҚ®зӣ‘жҺ§зӯүпјүйқһеёёжңүз”ЁгҖӮ
  • еҮҸе°‘жңҚеҠЎеҷЁиҙҹиҪҪпјҡдёҺдј з»ҹзҡ„иҪ®иҜўпјҲpollingпјүжҲ–й•ҝиҪ®иҜўпјҲlong pollingпјүзӣёжҜ”пјҢWebSocket еҮҸе°‘дәҶдёҚеҝ…иҰҒзҡ„ HTTP иҜ·жұӮпјҢд»ҺиҖҢеҮҸиҪ»дәҶжңҚеҠЎеҷЁзҡ„иҙҹиҪҪгҖӮ
  • дҝқжҢҒиҝһжҺҘзҠ¶жҖҒпјҡдёҖж—Ұ WebSocket иҝһжҺҘе»әз«ӢпјҢиҝһжҺҘдјҡдҝқжҢҒејҖж”ҫзҠ¶жҖҒпјҢзӣҙеҲ°е®ўжҲ·з«ҜжҲ–жңҚеҠЎеҷЁе…ій—ӯиҝһжҺҘпјҢиҝҷж„Ҹе‘ізқҖдёҚйңҖиҰҒжҜҸж¬ЎйҖҡдҝЎйғҪйҮҚж–°е»әз«ӢиҝһжҺҘгҖӮ
  • иҪ»йҮҸзә§еҚҸи®®пјҡWebSocket еҚҸи®®жң¬иә«зӣёеҜ№з®ҖеҚ•пјҢеӨҙйғЁејҖй”Җиҫғе°ҸпјҢиҝҷдҪҝеҫ—ж•°жҚ®дј иҫ“жӣҙеҠ й«ҳж•ҲгҖӮ
  • ж”ҜжҢҒдәҢиҝӣеҲ¶ж•°жҚ®пјҡWebSocket дёҚд»…ж”ҜжҢҒж–Үжң¬ж•°жҚ®пјҢиҝҳж”ҜжҢҒдәҢиҝӣеҲ¶ж•°жҚ®пјҢиҝҷеҜ№дәҺйңҖиҰҒдј иҫ“еӣҫеғҸгҖҒйҹійў‘гҖҒи§Ҷйў‘зӯүдәҢиҝӣеҲ¶ж•°жҚ®зҡ„еә”з”Ёйқһеёёжңүз”ЁгҖӮ

зјәзӮ№пјҡ вҖӢ

  • жөҸи§ҲеҷЁж”ҜжҢҒпјҡе°Ҫз®ЎзҺ°д»ЈжөҸи§ҲеҷЁжҷ®йҒҚж”ҜжҢҒ WebSocketпјҢдҪҶдёҖдәӣиҫғж—§зҡ„жөҸи§ҲеҷЁеҸҜиғҪдёҚж”ҜжҢҒпјҢиҝҷж„Ҹе‘ізқҖдҪ еҸҜиғҪйңҖиҰҒиҖғиҷ‘е…је®№жҖ§й—®йўҳгҖӮ
  • жңҚеҠЎеҷЁе®һзҺ°еӨҚжқӮпјҡзӣёжҜ”дәҺеҹәдәҺ HTTP зҡ„жңҚеҠЎпјҢе®һзҺ° WebSocket жңҚеҠЎеҷЁйңҖиҰҒжӣҙеӨҡзҡ„йҖ»иҫ‘жқҘеӨ„зҗҶиҝһжҺҘзҡ„з»ҙжҠӨгҖҒж¶ҲжҒҜзҡ„е№ҝж’ӯзӯүгҖӮ
  • йңҖиҰҒйўқеӨ–зҡ„йҖ»иҫ‘пјҡWebSocket жң¬иә«е№¶дёҚжҸҗдҫӣж¶ҲжҒҜзҡ„еҸҜйқ дј иҫ“пјҢеҰӮжһңйңҖиҰҒзЎ®дҝқж¶ҲжҒҜзҡ„еҸҜйқ жҖ§пјҢејҖеҸ‘иҖ…йңҖиҰҒиҮӘе·ұе®һзҺ°йҮҚдј жңәеҲ¶гҖӮ
  • и·Ёеҹҹй—®йўҳпјҡWebSocket еҗҢж ·еҸ—еҲ°еҗҢжәҗзӯ–з•Ҙзҡ„йҷҗеҲ¶пјҢи·Ёеҹҹ WebSocket иҝһжҺҘйңҖиҰҒжңҚеҠЎеҷЁж”ҜжҢҒзӣёеә”зҡ„ CORSпјҲи·Ёжәҗиө„жәҗе…ұдә«пјүи®ҫзҪ®гҖӮ
  • иҝһжҺҘз®ЎзҗҶпјҡWebSocket иҝһжҺҘйңҖиҰҒжңүж•Ҳз®ЎзҗҶпјҢеҢ…жӢ¬иҝһжҺҘзҡ„е»әз«ӢгҖҒдҝқжҢҒгҖҒйҮҚиҝһе’Ңе…ій—ӯзӯүпјҢиҝҷеҸҜиғҪдјҡеўһеҠ еә”з”Ёзҡ„еӨҚжқӮжҖ§гҖӮ
  • иҙҹиҪҪеқҮиЎЎеҷЁзҡ„ж”ҜжҢҒпјҡеңЁдҪҝз”Ё WebSocket ж—¶пјҢиҙҹиҪҪеқҮиЎЎеҷЁйңҖиҰҒиғҪеӨҹеӨ„зҗҶй•ҝиҝһжҺҘпјҢиҝҷеҸҜиғҪдјҡйҷҗеҲ¶жҹҗдәӣдә‘жңҚеҠЎжҲ–д»ЈзҗҶжңҚеҠЎеҷЁзҡ„дҪҝз”ЁгҖӮ

еңЁжҜ”иҫғжҖ§иғҪж—¶пјҢйңҖиҰҒиҖғиҷ‘еӨҡдёӘж–№йқўпјҢеҢ…жӢ¬жү§иЎҢйҖҹеәҰгҖҒеҶ…еӯҳдҪҝз”ЁгҖҒе…је®№жҖ§гҖҒжҳ“з”ЁжҖ§е’ҢеҠҹиғҪзү№жҖ§гҖӮдёӢйқўжҳҜдёҖдёӘз®ҖеҢ–зҡ„иЎЁж јпјҢжҰӮиҝ°дәҶ XMLHttpRequestгҖҒFetch APIгҖҒAxios е’Ң WebSocket иҝҷеӣӣз§Қж–№ејҸзҡ„жҖ§иғҪжҜ”иҫғпјҡ

зү№жҖ§/ж–№ејҸXMLHttpRequestFetch APIAxiosWebSocket
жү§иЎҢйҖҹеәҰдёӯзӯүеҝ«еҝ«йқһеёёеҝ«пјҲй•ҝиҝһжҺҘпјү

е…је®№жҖ§ж—§жөҸи§ҲеҷЁеҸҜиғҪйңҖиҰҒ ActiveXObjectж–°жөҸи§ҲеҷЁж–°жөҸи§ҲеҷЁпјҢж—§жөҸи§ҲеҷЁйңҖиҰҒ polyfillж–°жөҸи§ҲеҷЁпјҢж—§жөҸи§ҲеҷЁйңҖиҰҒ polyfill
жҳ“з”ЁжҖ§еӨҚжқӮзҡ„ APIпјҢеҹәдәҺеӣһи°ғз®ҖжҙҒзҡ„ APIпјҢеҹәдәҺ Promiseз®ҖжҙҒзҡ„ APIпјҢеҹәдәҺ PromiseпјҢжҸҗдҫӣжӢҰжҲӘеҷЁз®ҖеҚ•зҡ„ APIпјҢдҪҶйңҖиҰҒеӨ„зҗҶиҝһжҺҘз®ЎзҗҶе’Ңж¶ҲжҒҜж јејҸ
еҠҹиғҪзү№жҖ§дәӢд»¶й©ұеҠЁпјҢж”ҜжҢҒеҗҢжӯҘиҜ·жұӮеҹәдәҺ PromiseпјҢдёҚж”ҜжҢҒеҗҢжӯҘиҜ·жұӮдё°еҜҢзҡ„й…ҚзҪ®пјҢжӢҰжҲӘеҷЁпјҢиҮӘеҠЁиҪ¬жҚў JSONе®һж—¶еҸҢеҗ‘йҖҡдҝЎпјҢдёҚж”ҜжҢҒ HTTP иҜ·жұӮж–№жі•

жңҖдҪідҪҝз”ЁеңәжҷҜ вҖӢ

  • XMLHttpRequest (XHR): йҖӮз”ЁдәҺйңҖиҰҒе№ҝжіӣе…је®№жҖ§зҡ„иҖҒж—§зі»з»ҹжҲ–йңҖиҰҒдёҺеӨҡз§ҚеҗҺз«Ҝзі»з»ҹдәӨдә’зҡ„еңәжҷҜгҖӮ
  • Fetch API: йҖӮз”ЁдәҺзҺ°д»ЈжөҸи§ҲеҷЁдёӯпјҢйңҖиҰҒз®ҖжҙҒиҜӯжі•е’Ңй“ҫејҸи°ғз”Ёзҡ„еңәжҷҜгҖӮ
  • Axios: йҖӮз”ЁдәҺйңҖиҰҒеңЁйЎ№зӣ®дёӯиҝӣиЎҢеӨ§йҮҸHTTPиҜ·жұӮпјҢдё”йңҖиҰҒдё°еҜҢй…ҚзҪ®е’ҢеҸ–ж¶ҲиҜ·жұӮеҠҹиғҪзҡ„еңәжҷҜпјҢеҰӮдёҺеҗҺз«Ҝзі»з»ҹдәӨдә’йў‘з№Ғзҡ„еҚ•йЎөеә”з”ЁпјҲSPAпјүгҖӮ
  • WebSocket: йҖӮз”ЁдәҺйңҖиҰҒе®һж—¶дәӨдә’ж•°жҚ®зҡ„еңәжҷҜпјҢеҰӮе®һж—¶иҒҠеӨ©еә”з”ЁгҖҒеңЁзәҝжёёжҲҸзӯүгҖӮ

жҖ»з»“пјҡеңЁйҖүжӢ©еүҚз«Ҝж•°жҚ®иҜ·жұӮж–№ејҸж—¶пјҢеә”ж №жҚ®йЎ№зӣ®зҡ„е…·дҪ“йңҖжұӮгҖҒе…је®№жҖ§иҰҒжұӮд»ҘеҸҠжҖ§иғҪиҖғиҷ‘жқҘеҶіе®ҡдҪҝз”Ёе“Әз§Қж–№жі•гҖӮжҜҸз§Қж–№жі•йғҪжңүе…¶дјҳзӮ№е’ҢеұҖйҷҗжҖ§пјҢзҗҶи§Је®ғд»¬зҡ„зү№зӮ№пјҢеҸҜд»Ҙеё®еҠ©ејҖеҸ‘иҖ…жӣҙеҘҪең°жһ„е»әй«ҳж•ҲгҖҒзЁіе®ҡзҡ„еүҚз«Ҝеә”з”ЁгҖӮ
