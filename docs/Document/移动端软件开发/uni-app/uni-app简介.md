# uni-app 简介

uni-app 是 DCloud 公司开源的一个基于 Vue 的跨多平台移动应用开发框架，它可以使用 Vue 来编译成 iOS、Android、H5、小程序等多个平台的应用。 uni-app 的优点是覆盖范围广，开发成本低，支持使用原生组件和插件，支持热更新等功能。 uni-app 的缺点是性能和体验不如原生，需要适配不同平台的差异，还有一些兼容性和稳定性的问题。
