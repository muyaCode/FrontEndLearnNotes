## MongoDB与其他数据库或库的字段对比

| MongoDB     | MySQL       | Mongoose |
|:-----------:|:-----------:|:--------:|
| database    | database    | database |
| collection  | table       | schema   |
| document    | row         | model    |
| field       | column      | entity   |
| index       | index       | methods  |
| primary key | primary key | statics  |
| 无           | table join  | virtual  |
