# 数据库简介

用来数据的存放和操作

## 数据库的种类

### 种类

关系型：Oracle、MySQL、SQLite 、SQL Server等

非关系型（**Not** Only **SQL**）：MongoDB（文档）、Redis/Memcache（内存）

### 关系型和非关系型数据库软件区别

相同点：都是数据库软件，用来存放项目数据

不同点：

``` bash
关系型：1.遵循SQL标准，换句话说语法大同小异、2.有库和表约束等
非关系型：1.没有统一标准、2.一般键值对形式存储、3.读取速度更快
```
