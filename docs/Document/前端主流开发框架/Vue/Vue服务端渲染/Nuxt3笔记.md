# Nuxt3笔记

Nuxt3 是一个基于 Vue3 的服务器端渲染（SSR）框架，它可以帮助你快速构建服务端渲染的 Vue 应用程序，开箱即用、更新快速，是一款可用于各种项目上面的优秀框架。

Nuxt3 的官网地址是：https://nuxt.com/，可以在这里了解 Nuxt3 的特性、文档、博客、社区等信息。

Nuxt3 的开源地址是：[https://github.com/nuxt/framework](https://nuxt.com/v3)，你可以在这里查看 Nuxt3 的源码、贡献者、许可证、问题、讨论等信息。

如果想了解更多关于 Nuxt3 的内容，你可以参考以下的一些资源：

- [Announcing 3.0 · Nuxt Blog](https://thecodingpro.com/courses/nuxt3)：这篇博客文章介绍了 Nuxt3 的发布情况，包括 Nuxt3 的特点、API 稳定性、浏览器和 Node.js 支持、社区贡献、未来计划等。
- [Nuxt3 高效入门全攻略 - TheCodingPro](https://nuxt.com/blog/nuxt3-rc)：这是一个 Nuxt3 的在线课程，涵盖了 Nuxt3 的基础知识、路由和布局、数据获取、SEO 和元标签、中间件、状态管理、过渡、错误处理、层、服务器路由、自动导入等内容。
- [Announcing Nuxt 3 Release Candidate · Nuxt Blog](https://nuxt.com/blog/nuxt3-rc)：这篇博客文章介绍了 Nuxt3 的候选版本，包括 Nuxt3 的新服务器引擎、混合渲染、模块化、部署、迁移等内容。
