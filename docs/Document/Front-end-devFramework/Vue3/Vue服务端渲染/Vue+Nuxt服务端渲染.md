# Vue+Nuxt服务端渲染

### Nuxt介绍：

- Nuxt是一个基于Vue的框架，它允许应用通过Node.JS在服务器进行渲染，而且拥有完全交互式JavaScript应用程序的所有优点。  

- 对于Vue开发者来说，Nuxt主要的优势是能够预先渲染视图，并将其作为静态文件提供。这对SEO优化有很大的效果，并在互动性方面有很大的提升。
