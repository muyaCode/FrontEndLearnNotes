1. 基于最新 `vue` 标准的：
   1. script setup 语法标准

   4. 最新 `vue` 全家桶
2. 基于大厂编程规范的：
   1. eslint
   2. prettier
   3. Commitizen
   4. husky
   5. commitlint
   6. pre-commit
   7. lint-staged
